# landingpage-eksamen

Dette projekt er en fiktiv produktwebsite for højttaleren **Lumina One**.  
Formålet er at præsentere produktet visuelt og funktionelt med fokus på design, brugervenlighed og enkel interaktion.

Projektet er lavet som en del af et skoleforløb og indeholder både landingpage, købs-side og login-side.

---

## Indhold
Projektet består af følgende sider:

- **Landingpage (index.html)**  
  Præsentation af produktet, farver, features og anmeldelser.

- **Købsside (buy.html)**  
  Produktvisning med farvevalg, specifikationer og “læg i kurv”-knap.

- **Login-side (login.html)**  
  Simpel login-side til visuel demonstration (ingen backend).

---

## Funktioner
- Farvevalg på købsiden med JavaScript  
  (billedet og titlen skifter, når man vælger en farve)
- Genbrug af header og footer på tværs af sider
- Responsivt layout til mobil og desktop
- Simpelt og læsevenligt HTML- og CSS-setup

---

## Teknologier
Projektet er lavet med:

- HTML5  
- CSS3 (Grid og Flexbox)
- JavaScript (DOM manipulation)
- Google Fonts (Poppins)

Der er **ingen backend** eller database – projektet er udelukkende frontend.
